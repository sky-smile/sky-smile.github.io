Windows Terminal美化
发表于 2019-11-28| 更新于 2019-12-01|Windows
字数总计: 833|阅读时长: 3 分钟

Windows Terminal是微软在Build 2019上推出了一款面向Windows10的命令行程序，这款程序集合了Windows上的PowerShell、CMD以及Windows Subsystem for Linux于一身，现已上架Microsoft Store，体验非常不错。
