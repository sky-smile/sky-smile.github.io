Markdown 编辑器语法指南

Markdown 是一种轻量级标记语言， 可以快速的撰写帮助文档或是用于论坛上发表消息。

基本技巧

代码

如果你只想高亮语句中的某个函数名或关键字，可以使用 function_name() 实现

通常编辑器根据代码片段适配合适的高亮方法，但你也可以用 包裹一段代码，并指定一种语言
